# 简介
使用C语言编写的针对 SubC 可用的编译器， 以此来熟悉编译器在词法分析，语法分析，语义分析，优化操作层面上的工作原理，也会适当在原作者的代码中增加一些功能并为每个函数添加了友好的函数注释。目前来说，运行产生的汇编代码并不能生成目标文件，这可能是使用的汇编语法有误的原因。

# 工具
使用vscode进行代码撰写，cmake进行程序组织，gdb工具进行调试。
