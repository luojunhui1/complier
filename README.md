# 简介
    基本来说是根据仓库：https://github.com/DoctorWkt/acwj手敲了一遍代码，以此来熟悉编译器在词法分析，语法分析，语义分析，优化操作层面上的工作原理，也会适当在原作者的代码中增加一些功能并为每个函数添加了友好的函数注释。

# 工具
    使用vscode进行代码撰写，cmake进行程序组织，gdb工具进行调试。
